# Liquid Staking Protocol

A Clarity smart contract implementation of liquid staking for STX tokens on the Stacks blockchain.

## 📝 Description

This protocol allows STX holders to stake their tokens while receiving liquid sSTX tokens that can be used in DeFi applications. The protocol maintains a dynamic exchange rate between STX and sSTX to reflect staking rewards.
